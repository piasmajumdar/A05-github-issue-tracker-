    - 1️⃣ What is the difference between var, let, and const?
      - Var, let, and const are used to declare variables.
      - For const, we cannot redeclare, reassign. But mutation is possible only for arrays or objects.
      - For let, we cannot redeclare but we can reassign.
      - For var, we can redeclare and reassign. It is hoisted but give undefined value. 
    - 2️⃣ What is the spread operator (...)?
      - Spread operator spreads the values. It's used by three dot (...) 
      - We can pass an array of 3 element with … (spread) to a function, function will take 3 parameters.
      - Example- 
      - numbers = [1, 4, 6];
      - const add = (a,b,c) => a+b+c;
      - add(...numbers)

    - 3️⃣ What is the difference between map(), filter(), and forEach()?
      - Map: map() is used to transform each element of an array and returns a new array. Ex- array.map(e => e+1);
      - filter: filter() is used to select multiple elements based on condition and returns a new filtered array. Ex- array.filter(e => e== 1);
      - forEach: forEach() is used to loop through in an array and it doesn't return anything by default. Ex- array.forEach(e => console.log(e));

    - 4️⃣ What is an arrow function?
     -  Arrow function is used to write function shortly. It's introduced in ES6. Ex- const add = (a, b) => a + b;

    - 5️⃣ What are template literals?
      - Template literals are used to create string with backticks (``) where we can write function,variables inside backticks by using ${ }.

      - Example- 
      - const name = "Pias";
      - const age = 24;
